# 🎓 CRUD-SAMPLE 🎓 

Um CRUD simples - back-end/front-end.

## [Heroku](http://crud-sample-full.herokuapp.com/crud-sample)

##  🛠️ Tecnologias e Ferramentas

* Java 11
* Spring Boot
* JPA Hibernate
* PostgreSQL
* HTML
* CSS
* Bootstrap
* JQuery
* Ajax
* Eclipse IDEA
* Postman

## 🔎Funcionalidades

* Cadastro de Usuário
* Listar Todos Usuários
* Buscar Usuário por nome
* Alterar Cadastro de Usuário
* Deletar Usuário
